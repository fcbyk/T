## 算术运算符

- **加法** `+`、**减法** `-`、**乘法** `*`、**除法** `/`、**取余** `%`
- **自增** `++`、**自减** `--`
  - `++a`：先加再用
  - `a++`：先用再加

## 赋值运算符

复合赋值：`+=`、`-=`、`*=`、`/=`、`%=`

```kotlin
a += 5   // 等价 a = a + 5
```

## 比较运算符

- **相等** `==`（调用 equals() 比较内容）
- **引用相等** `===`（比较是否是同一个对象）
- **不等** `!=`、**大于** `>`、**小于** `<`、**大于等于** `>=`、**小于等于** `<=`

## 逻辑运算符

- **短路与** `&&`：两个都 true 才 true
- **短路或** `||`：有一个 true 就 true
- **逻辑非** `!`：取反

## 区间运算符

- `..`：闭区间 `1..5` → 1,2,3,4,5
- `until`：不包含结尾 `1 until 5` → 1,2,3,4
- `downTo`：递减区间 `5 downTo 1` → 5,4,3,2,1
- `step`：步长 `1..10 step 2` → 1,3,5,7,9

## in 运算符

判断元素是否在集合或区间中：

```kotlin
5 in 1..10    // true
5 !in 1..3    // true
```

## is 运算符

类型检查（类似 Java 的 instanceof）：

```kotlin
x is Int      // 是否是 Int
x !is String  // 是否不是 String
```

## 空安全运算符

- `?.` 安全调用：`name?.length`（null 时返回 null）
- `?:` Elvis 运算符：`name ?: "default"`（null 时使用默认值）
- `!!` 非空断言：`name!!.length`（null 时抛异常）

## 位运算符

Kotlin 使用函数而非符号：

- **与** `and`、**或** `or`、**异或** `xor`
- **左移** `shl`、**右移** `shr`、**无符号右移** `ushr`
- **取反** `inv()`

```kotlin
5 and 3   // 与运算 -> 1
5 or 3    // 或运算 -> 7
5 shl 1   // 左移一位 -> 10
```
