# Demo3 字符类 & 范围匹配
- 正则里的**字符类**，就是**一次性匹配「一组字符中的任意一个」**
- **范围匹配**是字符类的简化写法，专门用来匹配连续的字符区间。

## 一、什么是字符类？
语法：`[]` 方括号
作用：**匹配括号内的任意一个字符**，只匹配**单个字符**。

### 基础示例
```
[abc]
```
含义：匹配 **a**、**b**、**c** 中的**任意一个字符**。

✅ 匹配成功：`a` `b` `c`
❌ 匹配失败：`d` `ab` `1`

### 更多例子
| 正则 | 含义 | 能匹配 |
|------|------|--------|
| `[123]` | 匹配 1、2、3 之一 | `1` `2` `3` |
| `[xyz]` | 匹配 x、y、z 之一 | `x` `y` `z` |
| `[,.!?]` | 匹配标点之一 | `,` `.` `!` `?` |

## 二、范围匹配（字符类的简化版）
语法：`[起始-结束]`
作用：匹配**连续区间内的任意一个字符**，不用一个个罗列。

### 最常用 3 种范围
1. **数字**：`[0-9]`
   匹配 0 1 2 ... 9 任意一个数字

2. **小写字母**：`[a-z]`
   匹配 a~z 任意一个小写字母

3. **大写字母**：`[A-Z]`
   匹配 A~Z 任意一个大写字母

### 组合范围（超级常用）
| 正则 | 含义 |
|------|------|
| `[a-zA-Z]` | 所有大小写字母 |
| `[a-zA-Z0-9]` | 所有字母 + 所有数字 |
| `[0-9a-f]` | 十六进制字符（0-9,a-f） |


## 三、字符类里的「排除匹配」
语法：`[^...]` 方括号内开头加 `^`
作用：**匹配「除了括号里的字符」以外的任意一个字符**

### 示例
| 正则 | 含义 |
|------|------|
| `[^0-9]` | 匹配**非数字** |
| `[^a-z]` | 匹配**非小写字母** |
| `[^abc]` | 匹配不是 a、b、c 的字符 |


## 四、字符类内部的特殊规则（必看）
1. **大部分元字符在 `[]` 里失去特殊意义**
   - `.` `+` `*` `?` `()` 在字符类里就是**普通字符**
   - 例：`[.]` 只匹配小数点，不匹配任意字符

2. **`-` 只有在中间才是范围，在开头/结尾就是普通减号**
   - `[0-9]` 范围
   - `[-09]` 匹配 `-` `0` `9`

3. **`^` 只有在字符类**最开头**才表示“非”**
   - `[^a]` 非 a
   - `[a^]` 匹配 a 或 ^

## 五、超实用速查表
| 正则 | 含义 |
|------|------|
| `[abc]` | 匹配 a、b、c 之一 |
| `[^abc]` | 匹配不是 a、b、c 的字符 |
| `[0-9]` | 匹配一个数字 |
| `[a-z]` | 匹配一个小写字母 |
| `[A-Z]` | 匹配一个大写字母 |
| `[a-zA-Z0-9]` | 字母+数字 |
| `[^0-9]` | 非数字 |

### 总结
1. **字符类 `[]`**：匹配括号内**任意一个**字符
2. **范围 `[x-y]`**：匹配连续区间，简化写法
3. **排除 `[^...]`**：匹配除了这些以外的字符
4. 它们**永远只匹配单个字符**

# 预定义字符类（简写版字符类）
预定义字符类，就是把常用的 `[字符范围]` 提前起好**简写名字**，不用每次都写一长串，写起来更短、更清晰。

## 一、最常用 6 个预定义字符类
下面这些**等价于一个字符类**，都只匹配**单个字符**：

| 简写 | 等价写法 | 含义 |
|------|----------|------|
| `\d` | `[0-9]` | 数字（digit） |
| `\D` | `[^0-9]` | **非**数字 |
| `\w` | `[a-zA-Z0-9_]` | 单词字符：字母+数字+下划线 |
| `\W` | `[^a-zA-Z0-9_]` | **非**单词字符 |
| `\s` | `[ \t\r\n\f]` | 空白符：空格、制表、换行等 |
| `\S` | `[^ \t\r\n\f]` | **非**空白符 |

简单记：
- 小写：匹配某类字符
- 大写：匹配**相反**的

## 二、逐个拆开讲
### 1. 数字相关
- `\d` → 匹配 **0~9 任意一个数字**
  例：`a\d` 可以匹配 `a1` `a5` `a9`

- `\D` → 匹配**不是数字**的任意一个字符

### 2. 单词字符
- `\w`
  匹配：**大小写字母、数字、下划线 _**
  常用于匹配变量名、用户名、ID

- `\W`
  匹配：**符号、空格、中文、标点**等非单词字符

### 3. 空白字符
- `\s`
  匹配：空格、制表符 `\t`、换行 `\n`、回车 `\r` 等

- `\S`
  匹配：**任何看得见的字符**（不是空白就行）

## 三、和普通字符类混用
预定义字符可以**放进 `[]` 里一起用**：

- `[a-f\d]`
  等价 `[a-f0-9]` → 十六进制字符

- `[\w\s]`
  字母数字下划线 + 空白

## 五、快速记忆表
- `\d` 数字
- `\D` 非数字
- `\w` 字母数字下划线
- `\W` 非字母数字下划线
- `\s` 空白
- `\S` 非空白
