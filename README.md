# demo18

## 类与对象 
- `class` 关键字定义类
- 存储属性、计算属性（只读/读写）
- 实例方法、类型方法（static）
- 嵌套类可定义在函数内部

## 构造函数
- 指定构造函数：初始化所有属性
- 便利构造函数（convenience）：必须调用本类其他构造器
- 可失败构造函数（init?）：条件不满足返回 nil
- 必要构造器（required）：子类必须实现
- 构造器链式调用

## 继承
- 子类使用 `:` 继承父类
- `override` 重写父类方法
- `super` 调用父类实现
- 属性观察器（willSet/didSet）监控属性变化
- `final` 禁止继承或重写

## 协议
- `protocol` 定义接口规范
- 类、结构体、枚举都可遵循协议
- struct 修改自身需标记 `mutating`
- 协议继承：子协议继承父协议
- 多协议遵循：用逗号分隔

## 扩展
- `extension` 为已有类型添加功能
- 可添加计算属性、方法、构造器（仅 convenience）、嵌套类型
- 不能添加存储属性，不能 override
- 为协议提供默认实现
- 让已有类型遵循新协议

## 访问控制
- `open`：跨模块访问 + 允许继承/重写
- `public`：跨模块访问，不允许跨模块继承/重写
- `internal`（默认）：当前模块内可访问
- `fileprivate`：当前文件内可访问
- `private`：当前作用域内可访问

## 计算属性与属性观察器
- 存储属性：直接存储值
- 计算属性：不存值，访问时现算（get/set）
- 属性观察器：willSet（赋值前）、didSet（赋值后）
- 延迟存储属性（lazy）：首次访问时初始化

## 多态
- 父类类型指向子类对象
- 同一方法调用，不同对象表现不同
- 编译时看类型，运行时看实际对象
- 实现条件：继承 + 重写 + 父类引用指向子类实例

## 结构体 vs 类
- 结构体：值类型（栈），自动 memberwise 构造器
- 类：引用类型（堆），需手动定义构造器
- 值类型赋值是拷贝，修改互不影响
- 引用类型赋值是引用，修改相互影响
- 结构体适合简单数据容器
