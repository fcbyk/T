# demo4 边界匹配符 (锚点)
边界匹配**不匹配任何实际字符**，只匹配**位置**。
就像给字符串“打标记”，规定：必须在**开头、结尾、单词旁边**才生效。

## 一、四个核心边界符号
| 符号 | 含义 | 位置 |
|------|------|------|
| `^` | 行/字符串开头 | 最前面 |
| `$` | 行/字符串结尾 | 最后面 |
| `\b` | 单词边界 | 单词与非单词之间 |
| `\B` | 非单词边界 | 单词内部、非单词内部 |

## 1. `^` 开头匹配
匹配**整个字符串最开始的位置**。

例：
- `^abc`
  只能匹配以 `abc` **开头**的内容
  ✅ `abc123`
  ❌ `xabc`、`123abc`

## 2. `$` 结尾匹配
匹配**整个字符串最后面的位置**。

例：
- `\.jpg$`
  只能匹配以 `.jpg` **结尾**的内容
  ✅ `photo.jpg`
  ❌ `jpg.png`、`a.jpg.bak`

## 3. 组合：精确匹配整个字符串
`^内容$` = **必须从头到尾完全一样**

例：
- `^\d{6}$`
  必须是**整整6位数字**，不多不少
  ✅ `123456`
  ❌ `a123456`、`1234567`、`12345`

这是校验验证码、手机号、邮编的标准写法。

## 4. `\b` 单词边界
匹配**单词和非单词之间的位置**。

规则：
一边是 `\w`（字母/数字/下划线），
另一边**不是** `\w`，就是 `\b`。

例：
- `\bcat\b`
  匹配独立单词 `cat`
  ✅ `a cat`、`cat!`、`(cat)`
  ❌ `category`、`catdog`、`scat`

作用：**防止部分匹配、防止误杀单词**。

## 5. `\B` 非单词边界
和 `\b` 相反：
匹配**两个单词字符中间** 或 **两个非单词字符中间**。

例：
- `\Bat\B`
  匹配 `at` 在**单词中间**
  ✅ `cat`、`matter`
  ❌ `at`、`at home`

## 二、超清晰对比示例
### ① 开头结尾
`^a` → 开头是a
`a$` → 结尾是a
`^a$` → 只有一个a

### ② 单词边界
`\bhello\b`
- 匹配：`hello`、`hello world`
- 不匹配：`helloworld`、`ohello`

## 三、一句话总结
- `^` 开头
- `$` 结尾
- `^...$` 精确完整匹配（校验专用）
- `\b` 单词边界（找整词）
- `\B` 单词内部
