# demo116 Rust 集合类型

两大集合 `Vec`、`HashMap`，以及它们的公共处理工具：迭代器。

## 知识点

| # | 知识点 | 说明 |
|---|--------|------|
| 1 | Vec 基础 | vec! 创建、push/pop、索引/get 安全访问、三种遍历方式 |
| 2 | HashMap | insert/get、entry 插入、遍历、remove |
| 3 | 迭代器 | map/filter/chain/take/skip 适配器、sum/count/max 消费方法、enumerate |
| 4 | Vec 进阶操作 | 切片、sort/sort_by、retain 过滤删除、extend 拼接、truncate 截断 |

## 运行

```bash
rustc app1.rs && ./app1
rustc app2.rs && ./app2
rustc app3.rs && ./app3
rustc app4.rs && ./app4
```
