# demo111 Rust 基础语法速览

函数、控制流、运算符——各语言都有的东西，快速过一遍 Rust 的写法差异。

## 知识点

| # | 知识点 | 说明 |
|---|--------|------|
| 1 | 函数基础 | fn、参数与返回值、表达式 vs 语句、代码块表达式 |
| 2 | 闭包与嵌套函数 | \|args\| 语法、IIFE、捕获外层变量、嵌套 fn 的局限 |
| 3 | 高阶函数 | 函数指针、函数当参数、函数当返回值 |
| 4 | 控制流 | if 是表达式、loop 可返回值、for 由迭代器驱动 |
| 5 | 运算符 | 算术/比较/逻辑/位运算、没有 ++/--、范围 .. 与 ..= |
