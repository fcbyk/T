# demo110 Rust 基础数据类型

Rust 基础数据类型的语法概览，覆盖标量类型、复合类型、字符串和类型转换。

| # | 知识点 | 说明 |
|---|--------|------|
| 1 | 整数 | i/u 全家桶、进制字面量、`wrapping_add` 溢出处理 |
| 2 | 浮点 | f32/f64、INF/NAN、`NaN != NaN` |
| 3 | 布尔 | 逻辑运算、Rust 不做 truthy/falsy 转换 |
| 4 | 字符 | 4 字节 Unicode、码点、中文/emoji |
| 5 | 元组 | 解构、`.0` 索引、`()` 单元类型 |
| 6 | 数组 | `[T; N]` 定长、`[0; N]` 批量初始化 |
| 7 | 切片 | `&[T]` 借用视图、Range 语法 |
| 8 | 字符串 | `&str` vs `String`、所有权、字节数 vs 字符数 |
| 9 | 类型转换 | `as` 截断、`From` trait |
| 10 | 类型别名 | `type` 关键字 |
